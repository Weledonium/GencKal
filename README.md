# GencKal
Kullanıcıların fiziksel verilerine ve kişisel hedeflerine dayanarak günlük kalori ve makro besin ihtiyaçlarını geleneksel ve bilimsel algoritmalarla hesaplayan bir beslenme planlayıcısıdır. Temel amaç, kullanıcılara kendi diyet süreçlerini yönetebilecekleri, verilerini kaydedip takip edebilecekleri kişiselleştirilmiş bir web paneli sunmaktır.
